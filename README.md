# Freela-Money
  - This project helpful to management of the money win on the night how music  profressional.


## Features
 - Register income Transaction
 - Register outcome Transaction
 - Total balance money 
 - Search Transaction

## Techs
 - ViteJS
 - Styled Components
 - RadixUI
 - Json Server
 - React Hooks
  
## Run Project

 ```bash
  git clone this repository 
   cd repository
   npm i or yarn 
   npm run dev
   npm run test
 ```
